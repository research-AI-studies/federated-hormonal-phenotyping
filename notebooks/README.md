# Notebooks

Exploratory and figure-generating notebooks live here.

**Before committing**, clear all cell outputs (e.g. `jupyter nbconvert
--clear-output --inplace <notebook>.ipynb`). Outputs frequently embed data
previews, plots, and tables, none of which may enter the public repository.
Generated artefacts must be written to the git-ignored `outputs/` tree, never
inlined in tracked notebook outputs.
